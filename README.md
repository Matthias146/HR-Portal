# HR-Portal (Modern Angular Enterprise)

## Über das Projekt
Dieses Projekt ist ein Minimum Viable Product (MVP) für ein modernes HR-Dashboard. Das Ziel ist es, Personalern morgens direkt den wichtigsten Überblick zu geben: Wer ist heute im Büro, wer ist krank und wer hat Urlaub? 

## Tech-Stack & Fundament
Dieses Projekt wird von Tag 1 an nach modernen B2B-Standards aufgebaut. Wir nutzen:
- **Framework:** Angular 21
- **Reaktivität:** Signals & Zoneless Change Detection für maximale Performance
- **Architektur:** 100% Standalone Components (Keine veralteten NgModules)
- **Styling:** SCSS
